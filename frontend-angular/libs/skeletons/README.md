# skeleton

## Usage

```ts
import { SkeletonsModule } from '@eworkbench/skeletons';
```

## Implementation

Skeleton component:

```ts
@Component({
  template: `<eworkbench-page-skeleton></eworkbench-page-skeleton>`,
})
class PageSkeletonComponent {}
```

## Running unit tests

Run `npx nx test skeletons` to execute the unit tests.
